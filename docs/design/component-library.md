## Core Components (10 Essential UI Elements)

### 1. Button
**Variants:** Primary, Secondary, Danger, Ghost, Link
**States:** Default, Hover, Active, Disabled, Loading
**Sizes:** Small (32px), Medium (40px), Large (48px)

### 2. Input Field
**Variants:** Text, Number, Email, Password, Search, Textarea
**Features:** Label, helper text, error state, icons, validation

### 3. Select Dropdown
**Variants:** Single, Multi-select, Searchable, Grouped

### 4. Table
**Features:** Sortable, paginated, filterable, responsive card view on mobile

### 5. Modal / Dialog
**Sizes:** Small (400px), Medium (600px), Large (800px), Full-screen (mobile)

### 6. Card
**Variants:** Basic, Elevated, Clickable, Metric card

### 7. Badge / Status Indicator
**Colors:** Green (success), Yellow (warning), Red (danger), Blue (info), Gray (neutral)

### 8. Alert / Toast Notification
**Types:** Success, Error, Warning, Info
**Behavior:** Auto-dismiss (success), persistent (error)

### 9. Form
**Patterns:** Single-column, two-column, multi-step wizard

### 10. Loading States
**Variants:** Spinner, Skeleton screen, Progress bar
